# Linux Home Lab

## Overview

This repository documents my hands-on Linux administration lab environment built using Rocky Linux and VirtualBox. The project serves as a portfolio of practical system administration exercises completed while developing skills in Linux, networking, troubleshooting, and RHCSA exam objectives.

The purpose of this project is to develop hands-on experience with Linux system administration concepts commonly used by system administrators, cloud engineers, and cybersecurity professionals.

## Technologies Used

- Rocky Linux
- VirtualBox
- SSH
- Bash
- GitHub

## Skills Practiced

- User and Group Management
- File Permissions
- SSH Configuration
- Networking Basics
- Package Management
- Linux Command Line
- System Troubleshooting

| Lab | Topic | Skills Demonstrated |

| Lab 03 | Linux File Permissions | chmod, chown, Ownership |
